# 교통 통제 데이터 추가

`data.js`에 새 교통 통제 레코드를 추가합니다.

## 수행 작업

1. `data.js` 파일을 읽어 기존 데이터 구조 확인
2. 사용자에게 추가할 데이터 입력 요청 (없으면 예시 데이터 사용)
3. 기존 구조와 동일한 형식으로 새 레코드 추가
4. 추가된 레코드 내용 요약 출력

## 데이터 필드

- 날짜 (입력일, 차단일)
- 공사명, 방향, 위치
- 작업자, 차량, 장비
- 담당자, 시공사, 현장 관리자 연락처
- 차선 폐쇄, 시간 제한 정보

## 규칙

- 기존 데이터 구조(필드명, 형식)를 반드시 유지
- 날짜 형식: YYYY-MM-DD
- 추가 후 `data.js` 구문 오류 없는지 확인
